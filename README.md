# 🖼️ Image Gallery Project

This is a responsive Image Gallery website created using HTML, CSS, and JavaScript.

## 🚀 Features
- Responsive design for all screen sizes 📱💻
- Category filters (All, Nature, Flower, Other)
- Smooth hover effects and transitions 🎨
- Lightbox view for full-screen image preview
- Next/Previous navigation buttons

## 🛠️ Technologies Used
- HTML5
- CSS3 (Grid & Flexbox)
- JavaScript

## 🎯 Purpose
This project was created to improve my frontend development skills and learn how to build interactive UI components.
